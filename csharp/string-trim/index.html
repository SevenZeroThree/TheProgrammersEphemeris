C# String.Trim

March 8, 2019 | 1 minutes to read

There are 2 overloads to the Trim() method:

Method Description
Trim() Removes all leading and trailing whitespace from a given string.
Trim(Char[]) Removes all leading and trailing instances of the specified characters from a given string

Trim

The Trim() method removes all leading and trailing whitespace from a given string.

public string Trim();

Examples

using System;

public class Program
{
    public static void Main()
    {
        var testString = "   some string   ";
        testString.Trim();
    }
}

// Output
/*
 *  some string
 */

Trim(char[])

The Trim(Char[]) method removes all leading and trailing instances of the specified characters from a given string.

public string Trim(params char[] trimChars);

Examples

using System;

public class Program
{
    public static void Main()
    {
        char[] charsToTrim = { '*', ' ' };
        string message = "*** some string ***";
        Console.WriteLine(message.Trim(charsToTrim));
    }
}

// Output
/*
 *  some string
 */
