Strings.LTrim

Visual Basic - March 8, 2019 by Christopher

Leading whitespace can be removed from strings in Visual Basic using the LTrim method. Dim TestString As String = " Extra spaces" Dim TrimString As String TrimString = LTrim(TestString) ' Output ' Extra spaces

Strings.RTrim

Visual Basic - March 8, 2019 by Christopher

Trailing whitespace can be removed from strings in Visual Basic using the RTrim method. Dim TestString As String = "Extra spaces " Dim TrimString As String TrimString = RTrim(TestString) ' Output ' Extra spaces

Strings.Trim

Leading and trailing whitespace can be removed from strings in Visual Basic using the Trim method. Dim TestString As String = " Extra spaces " Dim TrimString As String TrimString = Trim(TestString) ' Output ' Extra spaces
